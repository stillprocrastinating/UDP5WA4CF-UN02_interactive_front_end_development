  1. Click the type of die desired.
    • The question-marks die allows you to input a custom die type to roll.
    • Keyboard inputs can be used instead of mouse clicks:
      1. 2 for a d2.
      2. 4 for a d4.
      3. 6 for a d6.
      4. 8 for a d8.
      5. 0 for a d10.
      6. d (for "dozen") for a d12.
      7. t (for "twenty") for a d20.
      8. h (for "hundred") for a d100.
      9. o (for "other") for a custom die.
  2. The result of the roll will show in the area called "current roll".
    • The roll result will also be added to the running total of all rolls made in the area called "cumulative roll".
    • The roll result will also be listed in the area called "previous rolls".
  3. The "save" button can be clicked to save the list of "previous rolls".
    • The edit button can be used to edit the name of the save.
    • The delete button can be used to delete the save.
  4. The "reset" button can be clicked to clear the roll results.
